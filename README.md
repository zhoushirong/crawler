# 邑抔司

一个综合类网站

### 初始化系统

``` html
cd crawler 

npm install

cd crawler/static 

npm install
```

### 生成数据表

``` sql
mysql

cd sql

./source/createtable.sql
```

### 爬虫抓取小说内容

```html

cd crawler/tools

node ./createXXBook

node ./createXXBookDirectory

node ./createXXBookChapter

```

### webpack编译

``` html
cd crawler

webpack -w
```


### 启动

cd publish

npm install

npm start
